Abstract:
This week on Twitch, we're going deep on aligning Amazon VPC Lattice with the AWS Load Balancer Controller (LBC) through the Kubernetes Gateway API-what it actually looks like to use Gateway API as the contract, with ALB for north-south and Lattice for east-west. We'll break down how the controllers map to the same abstractions, where the boundaries are, and what this unlocks for multi-cluster and multi-VPC designs. If we have time, we'll also touch on migrating from AWS App Mesh to Lattice for ECS workloads-what changes, what simplifies, and where Lattice fits as the longer-term service connectivity layer.
