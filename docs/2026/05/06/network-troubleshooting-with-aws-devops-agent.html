Abstract:
AWS DevOps Agent is your always-available operations teammate that investigates incidents and identifies operational improvements as an experienced DevOps engineer would, by learning your applications and their relationships, working with your observability tools, and correlating telemetry, code, and deployment data across all environments. In this session, we provide an overview of AWS DevOps Agent and then walk through real networking scenarios, demonstrating how the agent autonomously triages incidents, provides root cause analysis, and surfaces actionable findings by correlating CloudWatch alarms, CloudTrail events, VPC Flow Logs, and configuration data.
